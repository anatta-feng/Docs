# 设置行间距

```xml
android:lineSpacingExtra="float"
```

 或

```java
setLineSpacing(float add, float mult)
```

# 设置字间距

仅在5.0以上，以下不清楚

```xml
android:letterSpacing=“float”
```

 或

```java
setLetterSpacing(float letterSpacing)
```
