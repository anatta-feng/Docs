线形图是通过在多个点之间绘制线段来连接一系列点所形成的图形。这些点按其坐标(通常是`x`坐标)的值排序。线形图通常用于识别数据趋势。

R中的通过使用`plot()`函数来创建线形图。

# 语法

在R中创建线形图的基本语法是

```R
plot(v,type,col,xlab,ylab)
```

以下是使用的参数的描述 - 

- *v* - 是包含数值的向量。
- *type* - 取值`“p”`表示仅绘制点，`“l”`表示仅绘制线条，`“o”`表示仅绘制点和线。
- *xlab* - 是`x`轴的标签。
- *ylab* - 是`y`轴的标签。
- *main* - 是图表的标题。
- *col* - 用于绘制点和线两种颜色。
